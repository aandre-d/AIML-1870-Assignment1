# Assignment 1 Specification

## Project Overview
Create a minimal "Hello World" webpage and deploy it live to the internet using GitHub Pages.

## Requirements

### Core Requirements
1. **Name Display**: Page must display "Andre"
2. **Greeting**: Include a "Hello World" greeting message
3. **CSS Styling**: At least one CSS style applied (e.g., background color, font change, centered text)

### Technical Requirements
- Single HTML file (`index.html`)
- Embedded CSS in `<style>` tags
- Deployed via GitHub Pages

## Planned Features

### Layout
- Centered content box with frosted glass effect
- Name displayed above the greeting message
- Responsive design with proper viewport settings

### Styling
- **Background**: Purple gradient background (from #667eea to #764ba2)
- **Text Box**:
  - Semi-transparent background with backdrop blur
  - Colorful gradient border (pink, cyan, yellow)
  - Rounded corners with shadow
  - Padding for breathing room
- **Name Styling**:
  - Large, bold font (5rem)
  - Animated rainbow gradient effect that continuously cycles
- **Greeting Styling**:
  - Medium font size (3rem)
  - Text shadow for depth

### Stretch Challenge Features
- **Animated Fireworks**:
  - Continuous firework animations in random positions
  - Multi-colored particles (red, green, blue, yellow, magenta, cyan, white)
  - Particles explode outward and fade away
  - New firework every 0.8 seconds
  - Larger particles for better visibility

## Implementation Approach
1. Create HTML structure with semantic elements
2. Implement CSS for layout and styling
3. Add JavaScript for firework animations
4. Test locally
5. Deploy to GitHub Pages
6. Verify live deployment

## Success Criteria
- Page loads successfully on GitHub Pages
- All core requirements met
- Stretch challenge implemented
- Clean, readable code
- Visually appealing design
